Scénarios ChallengeJava01
--------------------
A.	
Scénarios nomminaux
--------------------
1.
The user inputs an insee numeral of a city in the entry field.
2.
The app checks the internet connection when the focus on the entry field is lost.
3.
the internet connection is tested with the url of API meteo-concept. the url is composed of a token and a correct insee numeral of a city which allows to get e server response.
4.
the api returns a response: a HTTP code 200.
5.
the api converts the JSON file receipted into a JSON object
6.
the app checks if the response is an objet which the type is a city. 
 The app returns only one city if the user has entered an insee's numerous of a city.
7.
if the response is a city, a jTable object is created to fill a JTable object of the application.
8.
the table is add to the frame of the application.
9.
the user consults the app with the displayed frame.
